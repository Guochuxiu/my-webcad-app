# 方案模板

## 使用时机

- 新需求
- 大于单文件修改的功能变更
- 涉及 `fscadweb` / `cadnginx` 落点判断
- 涉及交互、渲染、拾取、命令、性能、销毁路径

```md

- 当前问题
- 用户目标
- 约束条件

## 2. 源码基线

  - `...`
  - `...`
- 当前行为:
  - ...

- 放在 `fscadweb` 的部分:
  - ...

  - ...

  - ...
```

## WebCAD 专项补充项

方案里优先明确：

- Entity / Display 是否新增
- 命令是否新增，如何注册
- 对应 layer 是什么
- 是否涉及 selection、snap、gizmo、worker、batch、instance、LOD
- 删除和销毁路径是否安全
